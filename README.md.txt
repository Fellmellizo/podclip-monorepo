# PodClip Monorepo

## Estructura

- `frontend/` → React + Vite
- `backend/` → Express.js

## Cómo ejecutar

1. Abre dos terminales.
2. En la primera:

```bash
cd backend
npm install
node server.js
```
3. En la segunda:
```bash
cd frontend
npm install
npm run dev
```

Ahora puedes subir un archivo de **audio** junto con imágenes o un archivo **video** para generar clips automáticos.
